#### 2. Как запустить код?

Открыть терминал и выполнить последовательно команды:

```
git clone git@github.com:enfllome/keksobooking.git
cd keksobooking
npm install
npm run build
npm start
```

Команда клонирует репозиторий на ваш компьютер и подготовит всё необходимое для старта работы.
